How active is Sahitya's lifestyle?

This website presents a visual narrative of mobile data collected through Aware iOS Client. The following bar graph depicts the lifestyle of a user named Sahitya. The data is collected for over a period of 7 days starting from 10/10/2018 to 10/17/2018. Specifically, the table 'plugin_ios_activity_recognition' is used to collect meaningful information related to the lifestyle of the user.

Explanation of the above narrative

The above bar chart represents the distribution of various activities of the user on each day for a duration of 8 days. The data has four different kinds of meaningful activities, namely 'stationary', 'walking', 'running', 'automotive'. It is easily observable that the user has sedentary lifestyle with 'stationary' being the state for a large amount of time, even excluding the time for which the user sleeps, for about a third of each day. Since the sensor collects data about the activity type every 60 seconds, it makes sense to look at the distribution across various activities rather than the absolute numbers.
