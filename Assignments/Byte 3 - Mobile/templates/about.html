Accountability: About This Data Set

The data is collected through Aware iOS Client for a duration of 8 days starting from 10/10/2018 to 17/10/2018. The user has given access to readings from different sensors like accelerometer, battery, gps, activity recognition, etc. From the collected dataset, I found the sedentary lifestyle of the user to be noteworthy and hence plotted different kinds of activities the user is involved in on a particular day.

Where did this data come from?

The data is collected by the Aware iOS Client.

Who has access to it?

Only the primary user, myself, has access to it.
