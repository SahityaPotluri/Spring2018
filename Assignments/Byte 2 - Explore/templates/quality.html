Quality: About This Data Set

The dataset is collected from the ESPN CricInfo website, which is in an authentic source of information pertaining to Cricket. The ESPN makes sure to update the statistics frequently. This particular dataset was updated as recent as 9/9/2018. Hence the analysis we derive on this dataset would be coherent with the current world.

Is the data complete?

Yes, the data is complete. It does not have any missing valued entries and is also upto date. This data can answer the questions I attempt to answer, like 'Which wicket keeper has scored the highest number of dismissals in his career?' or 'Which player scored the largest number of catch dismissals?'

Is the data coherent?

Yes, the data is coherent. The number of dismissals acquired by a wicket keeper should be the sum of the dismissals via catches and stumpings. I found that sum of the catches and stumpings match with dismissals across all the players. I also verified that the type of values under each column is consistent with the expected type i.e. the Player column should comprise of strings, Span column should comprise the pattern {(year)-(year|Present)} like (1988-2012) or (1988-Present), and the other columns should comprise of integer values.

Is the data correct?

Yes. The data is correct. I verified this from three factors: 1) The entries in the dataset do not require a subjective evaluation, thereby reducing human bias or error 2) ESPN updates the data frequently and is a well know credible organization 3) I also cross-verified the values of top 20 entries with the values from Wikipedia and other cricket information related websites.
